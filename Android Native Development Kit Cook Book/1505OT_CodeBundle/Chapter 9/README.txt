Chapter9 contains the following code (in the sequence of appearing in the chapter):
1. PortingExecutable: an Android app demonstrates how to port a command line executable with Android NDK build system
2. PortingExecutableBuildSystem: an Android app deomonstrates how to port a command line executable with Android NDK standalone compiler
3. PortingExecutableAUI: an Android app demonstrates how to add a GUI for ported Android app
4. PortingExecutableAUIAsync: an Android app demonstrates how to use background processing and report progress to GUI at native code
Users need to download the boost library and compile the library by following the instructions in the book.
Usage: all the code are Eclipse projects. Simply use Eclipse import function to import the project.
