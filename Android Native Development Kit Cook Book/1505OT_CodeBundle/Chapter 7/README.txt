Chapter 7 contains the following code (in the sequence of appearing in the chapter):
1. JNIGraphics: an Android app demonstrates how to use jnigraphics at Android NDK
2. DynamicLinker: an Android app deomonstrates how to use Dynamic linker library at Android NDK
3. ZlibDemo: an Android app demonstrates how to use Zlib compression library at Android NDK
4. OpenSLESDemo: an Android app demonstrates how to use OpenSL ES Audio library at Android NDK
5. OpenMAXSLDemo: an Android app demonstrates how to use OpenMAX AL library at Android NDK

Usage: all the code are Eclipse projects. Simply use Eclipse import function to import the project.
