Chapter 4 contains the following code (in the sequence of appearing in the chapter):
1. TwoDG1: an Android app demonstrates how to draw 2D objects and OpenGL transform for Android NDK
2. CubeG1: an Android app deomonstrates how to draw 3D objects and OpenGL lighting at Android NDK
3. DiceG1: an Android app demonstrates how to map texture to 3D objects at Android NDK
4. CubeG2: an Android app demonstrates how to draw 3D objects in OpenGL ES 2.0 API at Android NDK
5. EGLDemo: an Android app demonstrates how to display OpenGL drawings with EGL API

Usage: all the code are Eclipse projects. Simply use Eclipse import function to import the project.
