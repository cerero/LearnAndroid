Chapter 5 contains the following code (in the sequence of appearing in the chapter):
1. NativeActivityOne: an Android app demonstrates how to implement a "pure" native activity at Android NDK
2. NativeActivityTwo: an Android app deomonstrates how to implement a "pure" native activity at Android NDK
3. NativeWindowManagement: an Android app demonstrates how to draw things at Android NDK
4. NativeInputs: an Android app demonstrates how to handle input events at Android NDK
5. nativesensors: an Android app demonstrates how to access sensors at Android NDK
6. NativeAssets: an Android app demonstrates how to manage assets at Android NDK

Usage: all the code are Eclipse projects. Simply use Eclipse import function to import the project.
